# NovelsWeb
【簡介】<br>
此專案實作一個小說閱讀APP，由於小說內容來自於網路，故只有提供部分章節內容，
此APP主要目的在於實作出資料庫、Servlet、Android Studio的資料傳送功能。

【第一版功能】<br>
1.實作帳號登入-初始頁面<br>
2.實作帳號的註冊-透過自訂義menu與Intent<br>
3.利用JSON格式傳輸資料到手機APP<br>
4.ListView顯示回傳的資料<br>
5.小說內容的上/下頁/回首頁切換<br>
